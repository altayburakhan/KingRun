# 👑 King Run

A hybrid hypercasual mobile game that combines **endless runner** and **merge strategy** mechanics into a single gameplay loop.

Players collect different unit classes while navigating an obstacle-filled level. At the end of each run, collected units are deployed onto a battlefield where they can be strategically merged into stronger units before combat begins.

---

# Gameplay

King Run consists of two connected gameplay phases.

## 🏃 Runner Phase

Players control the king through an obstacle-filled level while collecting different unit classes.

The collected units are carried into the battle phase, encouraging players to optimize their route and maximize the strength of their army before combat.

---

## ⚔️ Merge Battle Phase

After completing the runner level, gameplay transitions into a merge-based battle arena.

Players strategically place collected units onto the battlefield and merge identical units into stronger versions before the battle begins.

The outcome depends on both the player's performance during the runner phase and their strategic decisions in the battle phase.

---

# Core Features

- Hybrid Runner + Merge gameplay loop
- Strategic unit placement system
- Merge evolution mechanics
- Multiple playable unit classes
- Wave-based battle system
- Level progression
- Mobile-first game design
